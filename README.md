# 🛠️ FIXCI – Trusted Services Platform

## 📌 Project Overview
FIXCI is a simple web platform designed to connect people in Côte d’Ivoire with trusted local service providers such as electricians, plumbers, cleaners, mechanics, and more.

This project was developed as part of a GitHub collaboration assignment, focusing on teamwork, version control, and basic web development.

## 🌍 Purpose
The goal of FIXCI is to make it easier for users to:
- Find reliable service providers
- Book services quickly
- Access verified professionals
- Improve everyday problem-solving

## 💻 Technologies Used
- HTML5 – Structure of the website
- CSS3 – Styling and layout
- GitHub – Collaboration and version control

## 🤝 Collaboration Process
- The project was developed in a team using GitHub
- Tasks were divided between content, layout, and styling
- Each member contributed with meaningful commits
- GitHub was used to track changes and manage updates

## ⚠️ Challenges
- Understanding GitHub collaboration
- Managing commits without conflicts

### ✔️ Solution
We practiced committing regularly and used clear commit messages to keep the project organized.
